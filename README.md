# Stock Portfolio Tracker

## Project Overview
The Stock Portfolio Tracker is a simple Python application that calculates the total investment value of a stock portfolio based on user-input stock quantities and predefined stock prices.

## Features
- Accepts stock names and quantities from the user.
- Uses a hardcoded dictionary to store stock prices.
- Calculates investment value for each stock.
- Calculates the total portfolio investment value.
- Displays a portfolio summary.

## Technologies Used
- Python
- Dictionary
- Input/Output Operations
- Basic Arithmetic

## Stock Prices Used

| Stock | Price ($) |
|--------|----------|
| AAPL | 180 |
| TSLA | 250 |
| GOOGL | 140 |
| MSFT | 420 |

## How It Works
1. Enter the number of stocks.
2. Enter the stock name and quantity.
3. The program checks the stock price from the dictionary.
4. The investment value for each stock is calculated.
5. The total investment value is displayed.
